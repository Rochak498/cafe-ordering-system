QR-Based Café Ordering MVP

This prototype supports the core Project A workflow: customer menu access, digital order placement, order tracking, and staff order management.

Customer Menu Staff Order Queue

Customer flow

  1. Scan QR code or open the menu link.
  2. Browse available items.
  3. Place an order and receive an order code.
  4. Track progress using the order code.

Staff flow

  1. Open the staff order queue.
  2. Review incoming orders.
  3. Update order status.
  4. Monitor metrics in the dashboard.
