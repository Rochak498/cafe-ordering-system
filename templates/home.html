Customer workflow

  1. Open the menu through a QR code or direct link.
  2. Choose an available menu item.
  3. Place the order and receive an order code.
  4. Track progress using the order code.

Staff workflow

  1. Review incoming orders in the staff queue.
  2. Update the order status as work progresses.
  3. Use the dashboard to monitor operational metrics.
  4. Export order data for review if required.
